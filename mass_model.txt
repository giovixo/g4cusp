MASS MODEL RELEASE
The code contain a given number on branches. A branch can corresponds to a given release of the mass model, as for example "20240223_mass_model", or to a specifit model, as for example "toy_model". All the reports are in the main branch. 

IMPORTANT NOTE 
All the mass model releases, provided by Giovanni Lombardi, are avaliable in mass_model_gdml directory 
The active mass model is a modified version of a given release. These changes are mandatory to define all the materials and to have a correct detectors naming.
In the mass_model_gdml directory <cusp_payload_GEANT4-worldVOL-structure_release.xml> file contain the release, <cusp_payload_GEANT4-worldVOL-structure.xml> contains the active mass model.
